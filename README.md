⚡ Pokédex App

A web application developed with Angular 18, focused on basic login authentication, form validation, routing, and UI styling using Angular Material and Bootstrap.

🚀 Technologies used
Angular 18
Angular Material
Bootstrap 5
TypeScript
SCSS
AOS (Animate On Scroll)

🎯 Current features
Login with username and password
Reactive Forms validation
Navigation between routes
Basic route protection using guards (in use)
Custom login page styling
Animations using AOS
Session handling using localStorage
